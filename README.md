# bot-comunidade-sao-paulo
Esta skill de chatbot para IBM Watson Assistant automatiza o atendimento sobre serviços públicos em São Paulo. O sistema processa NLP para informar sobre o status das linhas de metrô/trem, trânsito em grandes avenidas, horários de UBS, centros de acolhida e vacinação. Utiliza lógica de entidades com fuzzy match para entender variações de busca.
